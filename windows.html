📦 安裝步驟（首次使用）

1

安裝 Visual C++ Build Tools

InsightFace / ONNX Runtime 需要 C++ 編譯器。下載 Build Tools，勾選「使用 C++ 的桌面開發」。

2

建立虛擬環境並安裝依賴

PS> python -m venv venv
PS> .\venv\Scripts\activate
(venv) PS> python -m pip install --upgrade pip
(venv) PS> pip install -r requirements.txt
3

下載活體檢測模型（可選）

(venv) PS> python scripts/download_models.py
4

驗證安裝

(venv) PS> python -c "import cv2; import insightface; import ultralytics; print('All OK')"

🔧 常見問題

相機畫面全黑
設定 → 隱私權與安全性 → 相機 → 開啟「允許桌面應用程式存取相機」
python 指令找不到
安裝 Python 時未勾選「Add Python to PATH」，請重新安裝或手動加入環境變數。
.\venv\Scripts\activate 被擋
以管理員身份執行：Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
中文顯示亂碼
修改 main.py 第 108 行字型路徑為 C:/Windows/Fonts/msjh.ttc
